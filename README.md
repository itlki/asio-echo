# Asynchronous Echo Server using Asio C++

Just to test [Asio C++](https://think-async.com/Asio/) out.

## Quick Start

### Server

```console
$ make
$ ./echo_server
```

### Client

You can just use `telnet` as a client

```
$ telnet localhost 6970
hello
hello
$
```
